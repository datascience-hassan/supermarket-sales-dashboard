# 🛒 Supermarket Sales Analysis & Dashboard

## Project Overview
This project analyzes supermarket transaction data from multiple branches to identify sales trends, customer behavior, product performance, and profitability insights. An interactive Excel dashboard was created to present the results and support business decision-making.

## Dataset Description
- **Time Period:** January to March 2019
- **Branches:** Yangon, Mandalay, Naypyitaw
- **Level:** Transaction-level sales data
- **Data Includes:** Customer demographics, product categories, prices, payment methods, revenue, profit, and customer ratings

## Key Questions Answered
- Which product lines generate the most sales?
- Which branches or cities perform the best?
- Do members spend more than normal customers?
- Which payment methods are used the most?
- What are the busiest shopping days and times?

## Dashboard Features
- Sales by product line
- Sales by branch and city
- Customer type analysis
- Payment method breakdown
- Profitability insights
- Customer rating analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Basic Data Cleaning

## Dashboard Overview
The dashboard provides insights into:
- Sales trends over time
- Top-performing product categories
- Member vs normal customer spending behavior
- Branch-level sales comparison
- Customer rating patterns

Filters allow users to explore the data by branch, city, product line, payment method, and customer type.
